# site-config notes

Key blocks your workflow should set:

- `brand`
- `layout`
- `seo`
- `images`
- `hero`
- `trustBar`
- `services`
- `about`
- `coverage`
- `testimonials`
- `faq`
- `cta`
- `footer`

## Example layout values

```json
{
  "layout": {
    "theme": "trade-blue",
    "headerVariant": "header-a",
    "heroVariant": "hero-b",
    "servicesVariant": "services-c",
    "aboutVariant": "about-a",
    "coverageVariant": "coverage-b",
    "testimonialsVariant": "testimonials-a",
    "faqVariant": "faq-b",
    "ctaVariant": "cta-c",
    "footerVariant": "footer-a",
    "copyStyle": "family-run-trust",
    "sectionOrder": ["hero", "trustBar", "services", "about", "coverage", "testimonials", "faq", "cta"]
  }
}
```
