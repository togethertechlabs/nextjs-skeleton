# Next.js Business Template

A reusable Next.js business website template powered by a single `site-config.json` file.

## Quick start

```bash
npm install
npm run dev
```

## Update site content

Edit `site-config.json`.

## Includes

- Next.js App Router
- TypeScript
- Tailwind CSS
- Reusable section components
- Health endpoint at `/api/health`
