# Unique Next.js Skeleton

Premium multi-page business template for AI-generated brochure sites.

## Includes

- Multi-page structure: home, about, services, service detail, coverage, area detail, contact
- Variant-ready section ordering and layout controls
- Typed `site-config.json`
- SEO metadata, sitemap, robots
- Image library folder structure with manual drop-in assets
- Safe fallbacks for generated content

## Image setup

Add your trade images into:

- `public/images/generic/`
- `public/images/plumbing/`
- `public/images/electrician/`
- `public/images/roofing/`
- `public/images/painter/`
- `public/images/plasterer/`
- `public/images/construction/`

Recommended filenames:

- `hero-1.jpg`
- `about-1.jpg`
- `service-1.jpg`
- `service-2.jpg`
- `service-3.jpg`

You can also keep SVG placeholders until you swap them manually.

## Run locally

```bash
npm install
npm run dev
```
